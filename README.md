## Project Description
SHOPEZ is a MERN stack e-commerce application designed to provide a seamless shopping experience. Users can browse products, manage their shopping carts, and view total order values with ease.

### Key Features
* **Product Catalog:** Interactive list displaying product names, prices, and sizes.
* **Shopping Cart:** Add items to a virtual cart with real-time total calculation.
* **Responsive Interface:** Built with Bootstrap for a clean mobile and desktop experience.
* **Order Summary:** Direct integration for order details and summaries.

## Tech Stack
* **Frontend:** React.js, Bootstrap
* **Backend:** Node.js, Express.js
* **Database:** MongoDB
